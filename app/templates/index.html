Cel Aplikacji:

Pobieranie opinii o produktach z serwisu Ceneo.pl i poddawanie ich analizie

Opis Działania Aplikacji:

Dla produktu, którego kod zostanie podany jako parametr, pobrana zostaje jego nazwa oraz wszystkie dostępne opinie o nim. Następnie aplikacja poddaje je analizie i tworzy dla każdego produktu 2 wykresy.

Zastosowane biblioteki:

  • BeautifoulSoup4
  • Requests
  • Json
  • Os
  • Pandas
  • Numpy
  • Matplotlib
  • Flask

Zastosowane podejścia:

Szkielet aplikacji tworzy Flask z wykorzystaniem szablonu Jinja, znacznie skracającym ilość kodu i czas pracy. Sercem aplikacji są aplikacje Python pobierające opinie i poddające je obróbce. Przy tworzeniu aplikacji wykorzystany został również Python Virtual Environment
